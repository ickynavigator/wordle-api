## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Have Rust installed

### Setup

- Clone the project to your environment

### Install

- Run `cargo build` to install dependencies

### Usage

- Run `cargo run` to start the tide server
